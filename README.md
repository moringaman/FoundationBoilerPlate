# FoundationBoilerPlate

### This Boilerplate uses the Meteor Foundation for sites package Foundation-sites 6.2.3

#### Elements Included

* Off Canvas Menu
* Top menu bar
* Fully Responsive
* Parallax elements
* Hero Image
* Font Awesome Icons
* Iron Routing

### Meant as a starting point for meteor apps using foundation

## Usage Instructions

1. Download or clone repo
2. install [Meteor](www.meteor.com) with command: **curl https://install.meteor.com/ | sh**
3. Run Meteor from app root directory
